# ✿ Portfolio

A personal portfolio website with:
— gamified achievements, sidebar navigation, project category tabs
— terminal aesthetic, dark moody hero, skill bars, blog section
— whimsical charm symbols, image-forward galleries, illustrative warmth

---

## 🗂 File Structure

```
portfolio/
├── index.html          ← Main HTML 
├── css/
│   └── style.css       ← All styles (colors, fonts, layout)
├── js/
│   └── main.js         ← Interactions, animations, achievements
├── assets/
│   ├── images/         ← Project screenshots 
│   └── resume.pdf      ← Resume PDF 
└── README.md
```
